二次剩余

发表于 2023-07-16 更新于 2023-09-02
二次剩余

问题引入

求解关于 \(x\) 的方程

\[x^2\equiv n \pmod p\]

其中 \(p\) 为一个奇质数，\(n,p\) 已知，求 \(x\)

解的数量

\(n\) 是二次剩余当且仅当原方程 \(x^2 \equiv n \pmod p\)有解。

上述方程无解的 \(n\) 称作非二次剩余

那么问题来了：

对于二次剩余 \(n\) ，\(x^2 \equiv n \pmod p\) 有多少解？

我们拎出其中的两个解 \(x_1,x_2(x_1 \neq x_2)\)，分别讨论一下。

题意有

\[x_1^2\equiv n \equiv x_2^2 \pmod p\]

那么，将 \(x_2^2\) 移到左边来，有

\[x_1^2-x_2^2 \equiv 0 \pmod p\]

用一下平方差公式，有

\[(x_1-x_2)(x_1+x_2) \equiv 0 \pmod p\]

因为这里的 \(x_1,x_2\) 都是在模 \(p\) 意义下的解，所以 \(x_1,x_2\) 必定小于 \(p\)，故有

\[(x_1-x_2) \bmod p\neq 0\]

所以，我们有

\[x_1+x_2 \equiv 0 \pmod p\]

所以说，\(x_1\) 在模 \(p\) 意义下与 \(x_2\) 互为相反数。

假设还存在一个解 \(x_3\)，那么 \(x_3\) 与 \(x_1,x_2\) 都要互为相反数，这与 \(x_1 \neq x_2\) 相矛盾，故原方程最多只有两个解。

欧拉准则

当

\[n^{\frac{p-1}{2}}\equiv 1 \pmod p\]

时 \(n\) 为二次剩余。

证明：

若 \(n\) 是二次剩余，则有

\[n^{\frac{p-1}{2}} \equiv (x^2)^{\frac{p-1}{2}} \equiv x^{p-1} \equiv 1 \pmod p\]

若 \(n^{\frac{p-1}{2}} \equiv 1 \pmod p\)，将 \(n\) 表示为 \(g^k\)，其中 \(g=\delta_p(n)\)。

那么有

\[g^{k\frac{p-1}{2}} \equiv 1 \pmod p\]

由于 \(g\) 是原根，必有

\[p-1|k\frac{p-1}{2}\]

也就是说 \(k\) 一定是偶数，那么有 \(x \equiv g^{\frac{k}{2}} \pmod p\)满足条件。

也就是说 \(n^{\frac{p-1}{2}} \equiv 1 \pmod p\) 与 \(n\) 是二次剩余是等价的。

求解

BSGS

假设 \(g\) 是 \(p\) 的原根，那么只需要求解 \(g^a\equiv n \pmod p\)，其中 \(x=g^{\frac{a}{2}}\)。

这里 \(a\) 一定会是一个偶数。

证明：

回到上面的式子，因为 \(n\) 是二次剩余，所以

\[(g^a)^{\frac{p-1}{2}}\equiv {(g^{p-1})}^{\frac{a}{2}} \equiv 1 \pmod p\]

根据费马小定理 \(g^{p-1}\equiv 1 \pmod p\)，所以可以知道 \(g^{\frac{a}{2}}\) 肯定是有意义的。

所以一个合法的解就是 \(x=g^{\frac{a}{2}}\)。

而这个 \(a\) 的求解可以使用 BSGS 在 \(O(\sqrt{p})\) 的复杂度里面求解。

Cipolla

我们先随机一个 \(a\)，使得 \(a^2-n\) 为非二次剩余，由于非二次剩余的数量接近 \(\frac{p}{2}\)，所以这个的期望次数大概是 \(2\) 次。

定义 \(\omega = \sqrt{a^2-n}\)，类比一下 \(i\) 的定义，这样以后所有的数都可以表示成 \(a\omega +b\) 的形式

那么 \((a + \omega)^{p+1} \equiv n\pmod p\)，考虑证明。

先证明几个引理

引理 1

\[\omega ^{p-1}\equiv -1 \pmod p\]

证明：

因为 \(\omega ^2\) 为非二次剩余，所以

\[(\omega^2)^{\frac{p-1}{2}}\equiv \omega^{p-1} \neq 1\pmod p\]

根据费马小定理有

\[(\omega^2)^{p-1}\equiv 1 \pmod p\]

所以我们有

\[\omega ^{p-1}\equiv -1 \pmod p\]

得证

引理 2

\[(A + B)^p \equiv A^p + B^p \pmod p\]

证明：

将 \((A+B)^p\) 展开可得

\[(A+B)^p=\sum\limits_{k=0}^p\binom{p}{k}x^{k}y^{p-k}\]

其中因为 \(p\) 是质数，所以在 \(\binom{p}{k}\) 中，只有 \(k=0,p\) 的时候 \(\binom{p}{k}\) 才不是 \(p\) 的倍数。

所以可以得到

\[(A+B)^p≡A^p+B^p \pmod p\]

得证。

现在证明上述结论：

\[(a + \omega)^{p+1} \equiv (a + \omega)^p (a + \omega) \equiv(a^p + \omega^p) (a + \omega)\]

\[\equiv (a^p + \omega^{p-1}\times \omega) (a + \omega) \equiv (a - \omega) (a + \omega) \equiv a^2 - \omega^2 \equiv n \pmod p\]

因为要求的是 \(n\) 的二次剩余，而 \(p+1\) 也恰好是偶数，所以我们得到了答案 \(x= (a+\omega)^{\frac{p+1}{2}}\)。

这个求出的解里面很可能带有 \(\omega\) 项，所以我们就来证明其中不含 \(\omega\)。

假设 \((a+\omega){\frac{p+1}{2}}\equiv xa+y\omega \pmod p\)

所以有

\[(xa+y\omega)^2\equiv n \pmod p\]

拆开之后得到

\[x^2a^2+y^2(a^2-n)+2xay\omega\equiv n \pmod p\]

将 \(2xay\omega\) 移到右边，\(n\) 移到左边，可得

\[x^2a^2+y^2(a^2-n) - n \equiv -2xay\omega \pmod p\]

因为式子左边的不含 \(\omega\) 项，右边含有 \(\omega\) 项，故 \(-2xay\equiv 0 \pmod p\)

所以 \(x\equiv 0 \pmod p\) 或者 \(y \equiv 0 \pmod p\)。

假设 \(x\equiv 0,y\neq 0 \pmod p\)，那么

\[y^2(a^2-n)\equiv n \pmod p\]

\[[y^2(a^2-n)]^{\frac{p-1}{2}}\equiv y^{p-1} \times (a^2-n)^{\frac{p-1}{2}} \equiv 1 \times (a^2-n)^{\frac{p-1}{2}} \equiv (\omega^2)^{\frac{p-1}{2}} \neq 1 \pmod p\]

即 \(y^2(a^2-n)\) 为非二次剩余，又 \(y^2(a^2-n)\equiv n \pmod p\)，即 \(n\) 为非二次剩余，这与 \(n\) 为二次剩余矛盾。

所以 \(y\equiv 0 \pmod p\)。
